# TeamPulse Bridge Docs

TeamPulse Bridge is a production-style event ingestion platform focused on reliability, observability, and secure integrations.

## Highlights

- Multi-source webhook ingestion (Slack, Teams, GitHub, GitLab)
- Strong signature verification and optional admin JWT controls
- Queue backend abstraction with `log` and `pubsub` implementations
- OpenTelemetry traces and Prometheus metrics
- CI/CD workflows for verify, smoke, and release
